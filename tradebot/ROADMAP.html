  2. Install Python 3.10+ if not already. Check: python --version. On Windows, download from python.org.

  5. Open Angel One account at angelone.in (free, 15 min KYC online). Then create a free SmartAPI app at smartapi.angelone.in. Enable TOTP in the Angel One mobile app. Note the base32 TOTP secret string — put it in settings.py.
  6. Once Zerodha + Shoonya confirm — fill all API credentials in config/settings.py. Set DATA_SOURCE = "angel". Run python -m scheduler.tasks --task data to fetch 3 years of real BankNifty futures data.
  7. Re-run full walk-forward backtest on real futures data: python run_backtest.py. This is the real gate check — results on actual market data will be significantly different from yfinance index data.
  8. Set up Telegram bot — open Telegram → @BotFather → /newbot → get token → message yourself once → get chat_id. Fill in ALERTS section in settings.py. Set enabled = True.
  9. Start paper trading: python main.py — runs during market hours 9:15–15:30. In a second terminal: streamlit run monitor/dashboard.py. Watch the live gate check dashboard.
  10. Run for 3 months minimum. The dashboard shows backtest vs paper divergence weekly. When all gate conditions are green, Engine B classifier trains automatically and you're ready for Phase 4 live trading.
